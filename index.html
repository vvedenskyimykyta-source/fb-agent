FB Agent

Стартовая страница проекта для GitHub Pages. Выберите одну из демонстраций и откройте её прямо из репозитория.

FB Agent — основной концепт Главная демонстрация агентского интерфейса с чат-панелью и четырьмя состояниями.
FB Agent Launch — запуск кампании Концепт интерфейса запуска кампании с 8 состояниями и визуальным потоком.
FB Agent Pipeline Поток запуска кампании и связанная панель с состояниями pipeline.
FB Agent Settings Экран настроек агента для управления параметрами кампании.
FB Agent Settings v2 Вторая версия экрана настройки с обновлённым дизайном.
Важно: GitHub Pages будет запускать страницы как статические файлы. Убедитесь, что все `.jsx`-файлы находятся в той же папке и доступны в репозитории.
